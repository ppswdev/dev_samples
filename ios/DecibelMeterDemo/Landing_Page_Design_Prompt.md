# Landing Page 设计提示词文档

**用途**: 用于Figma、Midjourney、设计AI工具生成Landing Page设计稿

---

## 🎯 设计任务

为iOS专业分贝测量仪应用设计一个现代化、专业的多语言Landing Page。

---

## 📱 应用基本信息

**应用名称**: Professional Decibel Meter / 专业分贝测量仪

**应用类型**: 专业级iOS分贝计和噪音剂量计

**目标用户**: 职业健康专家、环境工程师、声学工程师、科研人员、普通用户

**核心价值**: 符合IEC 61672-1国际标准的专业噪声测量解决方案

---

## 🎨 整体设计风格

### 设计关键词

- **Professional** (专业) - 体现工具的专业性
- **Precise** (精准) - 强调测量的准确性
- **Technical** (技术感) - 科技产品的现代感
- **Clean** (简洁) - 信息清晰，不杂乱
- **Trustworthy** (可信赖) - 建立信任感

### 视觉风格

- **扁平化设计 (Flat Design)** - 现代、简洁
- **大量留白** - 不拥挤，信息呼吸感强
- **清晰的层级** - 标题、副标题、正文层次分明
- **卡片式布局** - 功能模块使用圆角卡片
- **渐变和微妙阴影** - 增加深度和现代感

## 🎬 动效设计说明

### 页面加载动效

```
Hero区:
  - 标题: 从下向上淡入 (0.6s, delay 0.2s)
  - 副标题: 从下向上淡入 (0.6s, delay 0.4s)
  - 按钮: 从下向上淡入 (0.6s, delay 0.6s)
  - Mockup: 从右侧滑入 (0.8s, delay 0.4s)
```

### 滚动触发动效

```
功能卡片:
  - 逐个从下向上淡入
  - 每个延迟 0.1s
  - 使用 Intersection Observer

数字统计:
  - 滚动到视口时从0计数到目标值
  - 持续时间 2s

图表:
  - 动画绘制效果
```

### 悬停效果

```
卡片:
  - transform: translateY(-8px)
  - box-shadow 加深
  - 过渡: 0.3s ease-in-out

按钮:
  - transform: translateY(-2px)
  - box-shadow 增强
  - 背景色加深
  
链接:
  - 下划线从左到右出现
  - 颜色变化
```

### 点击/交互效果

```
按钮点击:
  - scale(0.95) 按下效果
  - 持续 0.1s

FAQ展开:
  - max-height 从 0 到 auto
  - opacity 从 0 到 1
  - 持续 0.3s ease-in-out
```
